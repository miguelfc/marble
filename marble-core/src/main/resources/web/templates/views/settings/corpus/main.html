Validation Corpus
{{validation.progress}}
{{validation.error}}
{{validation.success}}

The validation corpus is used to validate the results of the processors against a known set of results. This corpus needs to comply with a specific format for it to be used by marble. The syntax is shown below. 

"<polarity>";"<text to evaluate>"

Where <polarity> could be "1" (positive), "0" (neutral) or "-1" (negative).

You can download the most recent version and upload it using the form below.
